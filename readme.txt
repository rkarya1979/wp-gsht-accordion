Google Spreadsheet to Accordion

== Description ==

Google Spreadsheet to Accordion is the perfect solution for when you need your google spreadsheet data into accordion format. We are using google API for featching data. 


= Website =
http://www.indianbusybees.com/


== Installation ==

1. Upload 'wp-gsht-accordion' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to setting > google keys. update the google key.
4. You have shortcode and widget to use google spreadsheet data into accordion format.
